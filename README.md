# Project Documentation for Perfume E-Commerce Website

## Features
- User registration and login
- Product catalog with detailed descriptions
- Shopping cart functionality
- Secure checkout process
- Order tracking

## Tech Stack
- **Frontend:** React.js, Redux
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Styling:** CSS, Bootstrap

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/hedroxuu-wq/tree-perfume-ecommerce.git
   ```
2. Navigate to the project directory:
   ```bash
   cd tree-perfume-ecommerce
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```

## Usage Guide
- To use this application, navigate to the homepage where you can browse products.
- Add items to the cart and proceed to checkout to complete your purchase.
